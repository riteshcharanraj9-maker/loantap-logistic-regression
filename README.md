## 💳 Introduction

LoanTap is a financial technology company based in India that provides innovative and flexible loan products to individuals and businesses.  

The company leverages technology and data science to provide quick loan approvals, flexible EMI options, and transparent lending processes.

This case study focuses on analyzing borrower data to identify patterns in creditworthiness and to build a machine learning model that predicts the likelihood of loan default.

---

## 🔹 Business Objective

LoanTap aims to use **data-driven insights** to improve its credit underwriting process.

The goal of this project is to:

- Analyze borrower financial data
- Identify risk patterns in loan applicants
- Build a **Logistic Regression model** to predict loan default risk
- Provide insights to optimize loan approval decisions

---

## 🔹 Our Task

As a Data Analyst / Data Scientist working for LoanTap:

- Perform Exploratory Data Analysis (EDA)
- Understand borrower behavior patterns
- Build a Logistic Regression model
- Evaluate model performance
- Provide actionable insights for credit risk assessment

| loan_amnt | Amount of loan applied by borrower |
| term | Loan duration (36 or 60 months) |

| installment | Monthly payment amount |
| grade | LoanTap assigned loan grade |
| sub_grade | LoanTap assigned loan sub-grade |

| home_ownership | Borrower home ownership status |
| annual_inc | Borrower's annual income |
| verification_status | Income verification status |
| issue_d | Loan funding date |
| loan_status | Loan status (Target Variable) |
| purpose | Purpose of the loan |
| title | Loan title |
| dti | Debt-to-income ratio |
| earliest_cr_line | First credit line opening date |
| open_acc | Number of open credit accounts |
| pub_rec | Number of derogatory public records |
| revol_bal | Total revolving credit balance |
| revol_util | Credit utilization rate |
| total_acc | Total credit accounts |
| application_type | Individual or joint application |
| mort_acc | Number of mortgage accounts |
| pub_rec_bankruptcies | Number of bankruptcies |
| address | Borrower address |
